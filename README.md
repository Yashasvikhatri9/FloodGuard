# FloodGuard: AI-Powered Urban Flood Prediction

FloodGuard is a state-of-the-art urban flood prediction and monitoring platform. It leverages AI, real-time sensor data, and advanced analytics to provide early warnings and actionable insights for city officials and citizens.

## 🚀 Features

- **Real-time Monitoring**: Track rainfall and water levels across multiple city wards.
- **AI-Powered Predictions**: Advanced machine learning models predict flood risks before they happen.
- **Interactive Dashboard**: Visualize flood-prone areas and historical data on an interactive map.
- **Critical Alerts**: Receive instant notifications for high-risk situations.
- **Community Reporting**: Citizens can report local flooding to improve system accuracy.

## 🛠 Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Maps**: Leaflet, React-Leaflet
- **Charts**: Recharts
- **AI Integration**: Google Gemini API

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file based on `.env.example` and add your `GEMINI_API_KEY`.

4. Start the development server:
   ```bash
   npm run dev
   ```
